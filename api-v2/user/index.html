# 根据标签查找用户

GET /api/v2/users
1

查询参数：

参数 类型 描述
tag array 可选，标签 ID

例子：

/api/v2/users?tags[]=1&tags[]=3
1

上面的例子表示检索标签 ID 为 1 和 3 下面用户。

注意，这个接口来还有其他参数，轻查看旧文档网站接口参数
